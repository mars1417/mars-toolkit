### ⚠️ 用免费版会碰到的「天花板」

> <font size="4" color="#e67e22">**免费隧道每重启一次，URL就会变**</font>
>
> 你的cpolar每次断开重连，公网地址就换了。CRM/签到/API全断——你得手动去找新URL、改配置。半夜断一次你就得爬起来一次。

> <font size="4" color="#e67e22">**VPN节点不自动更新**</font>
>
> V2Box节点过期后你得手动换订阅、手动连。忘记=服务断掉。

> <font size="4" color="#e67e22">**没有统一入口**</font>
>
> 今天用这个URL，明天换那个，推送给家长/客户的链接永远在变。

**这些都是免费工具本身的限制，不是你的问题。** Mars Toolkit的免费版已经帮你做了90%，剩下这10%如果不想手动搞——看下面。
