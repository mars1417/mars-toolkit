# 桌面文件整理

## 用途
桌面文件散落影响视觉和效率。定期将文件归入8个分类文件夹，保持桌面干净。

## 文件夹体系
| 文件夹 | 用途 |
|--------|------|
| `_项目/` | 所有开发项目（乐吧运营系统/CRM/AI课等） |
| `_文档合同/` | 合同、法律文书、考试卷、方案文档 |
| `_知识库/` | 知识文档、API参考、提示词库 |
| `_素材视频/` | 图片、视频、海报素材 |
| `_归档/` | 老备份、不再使用的文件 |
| `_工具/` | 工具软件、快捷方式 |
| `_UI设计稿/` | 设计稿、原型 |
| `_快捷指令/` | Shortcuts |

## 分类规则
- 重复文件先去重
- 考试卷/合同 → `_文档合同`
- 法律文书（委托书/开庭策略）→ `_文档合同`
- 项目文档 → `_项目/相应子目录`
- 图片/截图 → `_素材视频`
- json备份 → `_归档`
- 临时文件直接删除

## 注意事项
- 不要移动正在使用中的项目文件
- `.DS_Store` 和 `.localized` 是系统文件，不动
- 乐吧运营系统（无_前缀）是用户常用项目入口，可保持
