## 7. 常见问题 FAQ

### Q1：我不会配置，买了能用吗？

**出海云：** 能。下单成功后页面上会直接给你一个链接，复制它，打开 V2Box → 点「导入订阅」→ 粘贴链接 → 自动连接。全程不需要手动配任何东西。

**买断工具：** 能。页面提供下载和一键安装脚本，下载后跑一下就行。

### Q2：支持什么客户端？

**出海云：** V2Box（macOS）、v2rayN（Windows）、Clash Meta、Quantumult X（iOS）、Shadowrocket（iOS）、Stash 等全部支持。

**买断工具：** macOS 平台，部分工具后续支持 Windows。

### Q3：速度怎么样？

优选 IP 自动选延迟最低的节点，日常 YouTube 4K 流畅，延迟 100-200ms。具体速度取决于你的本地网络。

### Q4：节点会失效吗？

系统自动维护节点池，始终保持可用。不会出现全部失效的情况。

### Q5：买了以后能退款吗？

所有产品支持 **3 天内无理由退款**。直接微信联系，秒退。

### Q6：包年送的 VPN Auto-Pilot 是什么？

一个 V2Box 配套工具，可以自动更新订阅、自动选最快的节点、断线自动重连。单买 ¥9.9，包年免费送。

### Q7：macOS Care Pack 真的是免费的吗？

是的，¥0，不要钱。下载就能用。以后可能会出付费 Pro 版，但基础版永远免费。

### Q8：VPN Auto-Pilot 和 Tunnel Guardian 有什么区别？

| | VPN Auto-Pilot | Tunnel Guardian |
|:-----|:--------------|:---------------|
| 管什么 | V2Box 节点轮换 | cpolar 隧道修复 |
| 解决 | 节点过期无人换 | 隧道重启 URL 变 |
| 适合 | 翻墙用户 | 内网穿透用户 |

### Q9：Pro Lite 和 Pro Max 什么时候出？

Pro Lite 预计 7 月底上线，Pro Max 预计 8 月上。关注官网或 GitHub 仓库第一时间获取。
