# 🛠️ Mars Toolkit · 火星工具包

**老旧设备的服务器化方案。**

在 **2015款MacBook Air（4GB内存、128GB SSD）** 上跑了 **7个生产服务6个多月** 的实战工具包。

## 快速开始

```bash
git clone https://github.com/mars1417/mars-toolkit.git
cd mars-toolkit
bash scripts/watchdog/install.sh
```
