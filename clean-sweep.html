🔧 技术说明

系统清理是一个 macOS 后台守护进程，基于 launchd 实现定时 + 事件触发双模式。默认每天凌晨2点自动执行一轮清理：先关闭WPS/Stocks等内存大户，再关闭Chrome空闲窗口释放图形内存，最后调用 sudo purge 释放非活跃页面。浏览器缓存清理支持Chrome和Safari。全部操作记录日志到 ~/.mars-toolkit/logs/。

❓ 常见问题

会把我重要的浏览器数据删掉吗？
不会。只清理 Cache 和 Code Cache 目录，不影响书签、密码、扩展、历史记录。Safari 同理，只清缓存目录。
sudo purge 安全吗？
非常安全。purge 是 macOS 内置命令，作用是强制释放非活跃内存页面。这些页面本来就是系统可以在需要时回收的，purge 只是让这个过程立即发生。不会导致任何程序崩溃或数据丢失。
4GB内存的旧Mac效果明显吗？
非常明显。实测在2017款MacBook Air 4GB上，一轮清理可释放约600-800MB可用内存。Safari开10个标签页不卡了，App切换不再转彩虹圈。
能自定义清理内容吗？
可以。配置文件 ~/.mars-toolkit/clean-sweep.conf 中可以设置定时时间、内存阈值、清理哪些浏览器缓存、是否清理系统日志和废纸篓。
