🔧 技术说明

实机看板是一个基于 Shell 的轻量级 HTTP 服务，通过 macOS 内置命令（system_profiler、vm_stat、powermetrics、df、uptime 等）采集系统数据，以 JSON 格式存储并通过嵌入式 Web 页面展示。每30秒自动刷新数据。所有代码纯 Shell 实现，不依赖 Python/Node.js 等运行时。日志保存在 ~/.mars-toolkit/logs/real-dashboard.log。

❓ 常见问题

手机也能看吗？
能。只要手机和Mac在同一个局域网，手机浏览器打开 http://你的MacIP:8080 就能看到所有数据。配合内网穿透工具可以从外网访问。
对性能有影响吗？
几乎无影响。数据采集脚本每30秒运行一次，每次耗时不到0.5秒。HTTP服务基于轻量级 nc 命令，几乎不占用CPU和内存。
能监控哪些数据？
CPU温度、CPU负载、内存使用量及百分比、磁盘使用情况、电池循环次数和健康状态、系统运行时间、进程总数。如有更多需求欢迎反馈，可定制扩展。
