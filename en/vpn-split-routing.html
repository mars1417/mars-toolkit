🔧 Technical Details

VPN Split Routing is a macOS network traffic management tool. It configures V2Box's leaf.conf with domain-level routing rules — development domains (DeepSeek, OpenRouter, GitHub API) go through the VPN tunnel, while domestic websites (Baidu, Taobao, Weibo) connect directly. Includes complete connectivity diagnostics, VPN interface detection, and config backup/restore. All operations run locally with no extra software required.

❓ FAQ

Does this only support V2Box?
Currently optimized for V2Box. But the connectivity test and VPN interface detection work with any VPN client. You can manually configure split routing for other clients using the detected domain IPs.
Do I need to restart my computer?
No. Just restart the V2Box client for the new rules to take effect. The script provides the restart command.
Can this mess up my network?
No. The script automatically backs up your original leaf.conf before making any changes. You can restore it anytime if needed.
