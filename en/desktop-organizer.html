🔧 Technical Details

Desktop Organizer is a macOS file automation tool. It classifies desktop files into 8 preset categories by matching file extensions against configurable rules. Uses a "copy-then-delete" safety strategy to prevent data loss. Fully customizable — add your own categories and file patterns. All operations run locally with no internet connection required.

❓ FAQ

Will it delete my files?
No. The tool uses a "copy then delete" safety strategy — even if the process is interrupted, no files are lost. System files (.DS_Store, etc.) are automatically skipped.
Can I customize the categories?
Absolutely. The CATEGORIES variable in the config file lets you define custom folder names and file extension mappings.
How do I undo the organization?
The tool doesn't delete original files — they're just moved to categorized folders. Simply drag them back to your desktop from the Finder.
