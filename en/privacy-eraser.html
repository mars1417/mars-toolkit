🔧 Technical Details

Privacy Eraser uses macOS built-in commands to clean various privacy traces. Browser cleaning targets Cache and Cookie directories, leaving bookmarks and passwords untouched. Recent documents are cleared via defaults commands for both system-level and per-app recent lists. Clipboard is cleared via pbcopy. All operations logged to ~/.mars-toolkit/logs/. Customize via ~/.mars-toolkit/privacy-eraser.conf.

❓ FAQ

Will this delete my bookmarks/passwords?
No. Browser cleaning only targets cache, cookies, and LocalStorage directories. Bookmarks, saved passwords, and browsing history are never touched. If worried, disable browser cleaning in config.
Is this enough before lending my Mac to someone?
Yes — it covers the three major privacy areas: browser cache, recent documents, and clipboard. For extra safety, enable system log and download history cleaning in config for complete coverage.
Can it auto-clean daily?
Yes. The installer includes a launchd service that runs daily at 3AM. Or run it manually anytime with one command.
Does it only clean my user directory?
Correct. All operations are within the current user's directory. System-level log cleaning requires sudo and is disabled by default.
